# DESAFIO TÉCNICO - API REST com Spring Boot
## Stack do Projeto

- Java 21
- Spring Boot 4.0.3
- Maven
- Spring Data JPA / Hibernate
- H2 - banco de dados em memória para desenvolvimento e testes
- Swagger(springdoc-openapi) — documentação interativa da API
- Lombok - redução de boilerplate (getters, setters, construtores)
- MapStruct - mapeamento entre entidades e DTOs
- JUnit 5 - testes unitários e de integração (com AssertJ e Mockito)
